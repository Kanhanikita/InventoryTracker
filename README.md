Ice Cream Ingredients Inventory Tracker
A web-based inventory tracking application built with Next.js and Firebase to help you manage and monitor ice cream ingredients. This app allows you to track your ingredients, update stock levels, and receive notifications for low inventory levels, ensuring you never run out of essential items.

Features
Ingredient Management: Easily add, update, and remove ingredients from your inventory.
Real-Time Syncing: All changes are synced in real-time across devices using Firebase's cloud database.
Low Stock Alerts: Set up thresholds for low stock levels and get notified when an ingredient runs low.
Search & Filter: Quickly search for ingredients or filter by categories for easy navigation.
User Authentication: Secure login system using Firebase Authentication for user access management.
Responsive Design: Fully responsive interface that works on desktops, tablets, and mobile dvices.

Technologies Used
Next.js: React-based framework for building server-side rendered and statically generated web applications.
Firebase: Backend-as-a-Service for real-time database, user authentication, and cloud storage.
Tailwind CSS: Utility-first CSS framework for styling the app.
JavaScript: Modern JavaScript features for writing clean, maintainable code.
